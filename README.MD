
# Word Count Go lang

Gives word frequency in a text in decreasing order

## API Reference

#### Get all items

```http
  GET /
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `` | `any` | Checks if APP is up and running |

#### Get item

```http
  POST /event
```

| Payload | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `Text`      | `text` | **Required**. Returns word frequency count |
